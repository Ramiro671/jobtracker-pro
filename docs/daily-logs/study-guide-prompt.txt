=== THE PROJECT STACK ===
Backend: .NET 10, C#, Clean Architecture (Domain/Application/Infrastructure/API)
Patterns: CQRS, MediatR, Repository Pattern, Unit of Work
Database: PostgreSQL (Neon.tech serverless) + Redis (caching)
ORM: Entity Framework Core with migrations
Auth: JWT (access token 60min) + Refresh Tokens (7 days) stored in DB
Validation: FluentValidation as MediatR pipeline behavior
Background Jobs: BackgroundService (StaleNotificationService - daily email)
Email: SMTP via System.Net.Mail.SmtpClient (disabled by default)
Rate Limiting: ASP.NET Core built-in (60/min general, 10/min auth)
Logging: Serilog (console + rolling file) + Application Insights
Testing: xUnit, Unit Tests + Integration Tests (WebApplicationFactory + InMemory EF)

Frontend: React 18, TypeScript, Vite, Tailwind CSS v4
State: Context API (AuthContext, ToastContext, ThemeContext)
HTTP: Axios with interceptor for automatic JWT refresh + request queuing
Routing: React Router v6 with ProtectedRoute
Features: Dark mode, CSV export, pagination, search, status filter, stale banner

DevOps: GitHub Actions CI/CD → Azure App Service (backend) + GitHub Pages (frontend)
